# Human-Reviewed Retrieval Evaluation Datasets

Place manually reviewed retrieval evaluation datasets here.

Do not store raw complaint text, audio files, or unmasked personal information. A valid dataset should contain only masked, structured complaint data and reviewed expected item keys.

Required reviewed fields include relevant VERIFIED_CASE item keys, CATEGORY_REFERENCE item keys, OFFICIAL_GUIDE item keys, expected evidence status, expected risk signal presence, no-match flag, and ambiguity flag.

Codex does not populate this directory with fake human-reviewed data.
